Explore Magazine
Layout Design
For this project, I created a mock travel brochure for Explore Magazine to inspire travellers to visit Scandinavia. Each spread features a different Scandinavian country, two must-see cities, a fun fact and a greeting in their language. For the page layouts, I kept some elements consistent such as headers, image placement and font size, yet played around with the text so that no spread was an identical replica.
